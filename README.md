# Bangladesh Badminton Federation

Official-facing web presence for the Bangladesh Badminton Federation.

## Stack

- Next.js (App Router) + React + TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger + `@gsap/react`
- Lenis smooth scroll
- Framer Motion (nav / micro-interactions)

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Notes

- Brand colors follow the BBF crest (orange + deep purple) on a formal warm-white ground.
- Imagery is placeholder stock — replace under `src/lib/images.ts` / `public/`.
- Logo: `public/images/bbf-logo.png`
